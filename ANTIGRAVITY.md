當您說 **「收工」** 或 **「下班了」** 時，我會執行以下流程：
1. **🛡️ 患者隱私與敏感資訊掃描 (PHI Security Scan)**：

2. **✍️ 自動備份與 Git 推送**：
   - 執行 `git add .`。
   - 根據我們今日的對話與修改內容（例如更新了教學網頁、文獻筆記或行政代辦），**自動生成極具專業度的語義化 Commit Message**。

3. **🧠 同步第二大腦知識庫**：
